# ✅ Todo List App

A clean, fully functional **Todo List** built with **React**, **HTML**, and **CSS** — no build tools or npm required. Just open `index.html` in any browser and start organizing your tasks.



---

*

---

## ✨ Features

- ➕ **Add tasks** — type and press `Enter` or click the Add button
- ✅ **Complete tasks** — click the checkbox to mark as done
- ✏️ **Inline editing** — double-click any task to edit it
- 🗑️ **Delete tasks** — remove tasks with the × button
- 🏷️ **Priority levels** — set High / Med / Low per task (color-coded)
- 🔍 **Filter view** — switch between All, Active, and Done
- 🧹 **Clear completed** — bulk-remove all finished tasks
- 📊 **Live counter** — shows remaining and total tasks

## ⚡ Getting Started

### Option 1 — Open directly in browser

```bash
# Clone the repo
git clone https://github.com/LuckyMaheshwari123/To-Do-List-App

# Open the file
cd todo-app
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

No installs needed. It works out of the box.

### Option 2 — Use a local server (optional)

```bash
# Using Python
python -m http.server 3000

# Using Node.js
npx serve .
```

Then visit `http://localhost:3000`
